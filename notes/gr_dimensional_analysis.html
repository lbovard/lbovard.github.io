Dimensional Analysis in General Relativity

This page contains discussions about dimensional analysis in General Relativity. Note that $G=1, c=1$ are implied unless otherwise specificed.

Dimensional arguments of the EFE

Being able to quickly do dimensional analysis in GR is important. Here are the key dimensional arguments for the terms in the EFE.

The metric must be dimensionless. This is because we know that are measuring $ds^{2}$ which has units of $L^{2}$ and the metric component are simply coefficients in front of the $dx^{i}dx^{j}$ terms. Thus the metric is always dimensionless

$$g \sim 1.$$

Now the Christoffel symbols involve $\sim g,\partial g$ and thus we immediately have that

$$\Gamma \sim L^{-1}.$$

The Riemann curvature tensor is a function of $\sim \partial \Gamma$ so thus we immediately have that

$$Riemann \sim L^{-2}.$$

Alternatively, the way to remember this is that $Riemann \sim \nabla_{[u}\nabla_{v]}$ where each covariant derivative introduces a $L^{-1}$ term.

It now follows immediately that the Ricci tensor and scalar are also $L^{-2}$ since they only involve contractions.

$$Ricci \sim L^{-2}.$$

Now using this information we can immediately recover the units of $\Lambda$ since it appears in the EFE

$$\Lambda \sim L^{-2}.$$

With this information we can also compute the Kretschmann scalar, $K\sim Riemann^{2}$, in Schwarzschild. The following is not a rigorous argument and merely a trick to remember the answer.

We recall that the $g_{tt}$ component, goes as $r^{-1}$ as $r\rightarrow 0$. Riemann is formed from second derivatives of $g_{tt}$, so we have that $Riemann \sim r^{-3}$ which leads to the Kretschmann behaving as

$$K \sim r^{-6}.$$
