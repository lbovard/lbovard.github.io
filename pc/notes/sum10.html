Digit sum

An often requested problem in programming contests is the digit sum. For most purposes, the following code works very well.

The algorithm is simply noticing that the last digit is always $a\mod10$ and then dividing by 10 to remove the last digit.


int digsum(int a) {
  // return the sum of the digits base 10
  int s=0;
  while(a>0) {
    s+=a%10; a/=10;
  }
  return s;
}
